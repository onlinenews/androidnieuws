Ubuntu Edge crowdfunded smartphone met Android én Ubuntu

22 juli om 17:20

Canonical, de mensen achter het open Linux-besturingssysteem Ubuntu, hebben de Ubuntu Edge aangekondigd. De eerste telefoon die de nieuwe mobiele versie van Ubuntu gaat draaien, naast Android.

Als je de telefoon vandaag nog bestelt, op de eerste dag van de crowdfunding via website Indiegogo, kost de Ubuntu Edge ‘slechts’ 600 dollar. In totaal moet er 32 miljoen dollar worden opgehaald. Morgen verandert die prijs en betaal je 830 dollar.

De Edge zal Android en Unbuntu draaien, twee mobiele besturingssystemen dus. Als je de telefoon via een connector aansluit op een beeldscherm, kan hij bovendien als volledige Ubuntu-pc functioneren, wat natuurlijk een handige bijkomstigheid is.

Ubuntu Edge heeft prima specificaties

Canonicals Edge huisvest een ‘snelle processor met meerdere cores’, 4GB aan intern geheugen, maar liefst 128GB aan interne opslagcapaciteit en een 4,5 inch-scherm met een resolutie van 1280 bij 720 pixels. Daarnaast bevat hij een 8-megapixel-camera en een 2-megapixel-frontcamera.

Die 600 dollar die je betaalt op deze eerste dag zijn echter wel zonder de verzendkosten, die alleen voor Amerikanen en Britten wegvallen. Als je morgen pas besluit om een Ubuntu Edge te nemen, betaal je dus maar liefst 830 dollar, wat natuurlijk een flinke prijs is.

De smartphone moet in mei 2014 op de markt verschijnen, en dat gebeurt alleen als die 32 miljoen dollar wordt opgehaald.
