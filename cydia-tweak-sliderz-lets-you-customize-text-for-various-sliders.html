Cydia Tweak Sliderz Lets You Customize Text For Various Sliders

Are you getting tired of the “Slide To Unlock” text on the lock screen? How about the “Slide To Answer” text? If so, Sliderz is a new Cydia tweak that will allow you to customize those sliders and much more.

Sliderz allows you to customize the unlock slider, power off slider, incoming call slider, and message slider. You also have the ability to disable to lock screen camera grabber.

Check out our video overview below:

If you can’t see the above video, please click this link.

If you’d like to customize one of the available sliders, tap on the corresponding text box and type in a new slider label. All of the changes will take effect immediately with no respring required. As I mentioned above, you can also disable the lock screen camera grabber. This will make your custom unlock text a bit bigger.

Overall, Sliderz is a very simple tweak, but I definitely recommend it if you’re looking to add some flair to iOS. Is Sliderz worth its price? In my opinion, it’s worth it to have all of these customizations in one place. Currently, Sliderz is available for $0.99 in the BigBoss repository.
