Rayman Fiesta Run komt op 7 november naar iOS en Android

25 oktober om 13:40

Platformgame Rayman Fiesta Run komt op 7 november naar iOS en Android. Dit maakt Ubisoft bekend via een YouTube-video.

Rayman Fiesta Run is het vervolg op het populaire Rayman Jungle Run. De game biedt vier nieuwe werelden met enkele nieuwe bazen. In totaal zijn er 75 levels en bevat het spel, evenals Rayman Legends, een Invasion-modus. Voor de mobiele versie is echter wel de tijdslimiet ingekort, waardoor het een race tegen de klok wordt.

Rayman Fiesta Run is ontwikkeld door Ubisoft Casablanca en draait op de UBIArt Framework engine, de techniek die ook gebruikt werd voor Rayman Legends. Een datum voor de release van de Windows Phone 8-versie werd nog niet genoemd.
