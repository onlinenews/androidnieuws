Microsoft brengt Wordament en Xbox Live-achievements naar Android

7 oktober om 13:34

Het woordspelletje Wordament, die aanvankelijk alleen voor Windows 8 en Windows Phone werd uitgebracht, is door Microsoft uitgebracht voor het Android-platform. Daarmee worden ook Xbox Live Achievements op Android beschikbaar gemaakt.

Wordament is een gratis game waarbij spelers verbindingen tussen letters kunnen maken om zo woorden te vormen. Het spel werd door Microsoft in eerste instantie uitgebracht op Windows 8 en Windows Phone. In december vorig jaar kwam er ook een iOS-versie beschikbaar. Inmiddels heeft Microsoft laten weten dat de game nu ook in Google Play te downloaden is, waarmee Wordament nu ook voor Android-gamers beschikbaar is. Daarmee is de game nu vrijwel crossplatform te spelen.

In Wordament is een koppeling met Xbox Live aanwezig voor het bijhouden van achievements. Om de game te kunnen spelen is wel een Facebook- of Xbox Live-account nodig; de Android-versie ondersteunt geen Google-accounts. Verder is een smartphone of tablet nodig die minimaal Android 4.0 draait.
