Microsoft brengt Remote Desktop-app uit voor Android en iOS

18 oktober om 15:55

Microsoft heeft voor de mobiele besturingssystemen iOS en Android een Remote Desktop-app uitgebracht. Met de applicatie kan de gebruiker een Windows-desktop via een smartphone of tablet benaderen.

De Remote Desktop-apps, die al door Microsoft subtiel waren aangekondigd, zijn gratis te downloaden via Google Play en de App Store van Apple. Voor het maken van een verbinding wordt Microsofts rdp-protocol gebruikt. Daarbij wordt tevens ondersteuning geboden voor RemoteFX, waardoor onder andere op afstand multitouch kan worden gebruikt om een Windows-machine te bedienen.

Microsoft claimt dat de Remote Desktop-verbinding voldoende is beveiligd door het toepassen van Network Level Authentication, een protocol dat onderdeel uitmaakt van rdp. Ook belooft de softwaregigant dat er op hoge kwaliteit beeld en geluid gestreamd kan worden door verbeterde compressiemethoden.

Om een verbinding met een Windows-pc te maken dient de machine logischerwijs wel te worden ingesteld als een host. Ook is een geschikte Windows-versie nodig: enkel de Enterprise- en Pro-versies van Windows 8 bieden Remote Desktop als host aan. Onder Windows 7 bieden Professional, Enterprise en Ultimate bestaat die mogelijkheid ook.
