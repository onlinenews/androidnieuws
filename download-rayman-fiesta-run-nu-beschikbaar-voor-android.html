Download: Rayman Fiesta Run nu beschikbaar voor Android

7 november om 14:26

Rayman Fiesta Run is vanaf vandaag te downloaden in Google Play. De platformgame is de opvolger van Rayman Jungle Run en borduurt verder op de gameplay die het vorige deel zo populair maakte.

Rayman Fiesta Run: heerlijke platformgame

Rayman Fiesta Run heeft tien levels meer dan zijn voorganger, wat het totaal op 80 brengt. Net als in Jungle Run rent Rayman automatisch door de levels heen, alhoewel jij hem langs diverse obstakels moet leiden door op het juiste moment op het scherm te drukken. Zo kun je het mannetje zonder ledematen laten springen, zweven, slaan en over muren laten rennen.

Daarnaast kan Rayman in ijslevels glijden en in muzieklevels wordt hij klein door middel van een trompet die hem opzuigt. Afhankelijk van het aantal lums dat je hebt behaald in Jungle Run ontvang je in Rayman Fiesta Run een extra portie lums, waarmee je bijvoorbeeld pakjes voor Rayman vrijspeelt.
